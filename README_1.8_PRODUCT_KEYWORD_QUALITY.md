# AutoCommerceAI 1.8 Product Keyword Quality

## 개선 내용
- 상품명 보완 입력 강화
- '쿠팡상품' 임시명 문제 완화
- 긴 상품명을 쇼핑쇼츠용 짧은 상품명으로 정리
- 타오바오 TOP10 검색어 자동 생성
- 도우인 TOP10 검색어 자동 생성
- 검색어별 목적/추천도 표시

## 예시
상품명:
주방 정리대 양념통 정리대 코너 주방 슬라이딩 수납장 다층 선반 조리도구 걸이 냄비뚜껑 거치대

자동 생성:
- 타오바오: 厨房 调料收纳架 抽拉式
- 도우인: 厨房 调料收纳架
- 댓글 키워드: 정리
- 콘텐츠 각도: 주방 정리 Before/After / 살림템 추천형
