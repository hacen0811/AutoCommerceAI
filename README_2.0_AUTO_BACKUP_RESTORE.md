# AutoCommerceAI 2.0 Auto Backup Restore

## 핵심 개선
- 프로젝트 JSON 자동 백업
- 백업/복원 메뉴 추가
- 새 버전에서 이전 폴더 자동 후보 감지
- DB가 없어도 JSON 백업으로 복구 가능
- 프로젝트 생성/수정 시 자동 백업 생성

## 메뉴
- 📦 데이터 가져오기
  - 이전 버전 폴더에서 DB와 영상 파일을 가져옵니다.
- 💾 백업/복원
  - 현재 프로젝트를 JSON으로 백업하거나 복원합니다.

## 권장 사용 방식
1. 작업 후 `💾 백업/복원`에서 백업 만들기
2. 새 버전 설치
3. `📦 데이터 가져오기` 또는 `💾 백업/복원`으로 복원
