# Sprint 1 - Real Vision

## 목표
영상 하나를 넣으면 아래 결과를 실제로 생성합니다.

- YOLO 객체 탐지 또는 Object AI 대체 분석
- PaddleOCR 자막 인식 또는 Vision AI 대체 분석
- Smart Cut 후킹 컷 추천
- 썸네일 후보 추천
- Caption Safe 자막 위치 추천
- Auto Editor CapCut 타임라인
- 쇼츠/릴스/인포크 문구

## 사용 순서
1. AI 모델 연결에서 설치 상태 확인
2. 프로젝트 생성
3. 영상관리에서 영상 업로드
4. Real Vision Runner 실행
5. 결과 확인
6. CapCut 편집

## 설치
YOLO/PaddleOCR 실제 실행은 선택 설치입니다.

```powershell
python -m pip install -r requirements_ai_install.txt
```
