# AutoCommerceAI 3.1 Menu Routing Fix

## 수정 내용
3.0에서 `쇼츠 스튜디오 3.0` 메뉴를 눌렀는데 `데이터 가져오기` 화면이 나오는 문제를 수정했습니다.

## 방식
기존 if/elif 메뉴 라우팅을 없애고 `ROUTES` 딕셔너리 기반으로 변경했습니다.

## 확인해야 할 메뉴
- 🎞️ 쇼츠 스튜디오 3.0 → Shorts Studio 화면
- 📦 데이터 가져오기 → 데이터 가져오기 화면
- 🧩 프로젝트 빠른 복구 → 프로젝트 복구 화면
- 💾 백업/복원 → 백업/복원 화면

## 실행
`RUN_APP.bat` 더블클릭
