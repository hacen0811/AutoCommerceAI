# AutoCommerceAI 2.1 Easy Start Recovery

## 핵심 개선
- PowerShell 가상환경 활성화 없이 실행 가능한 BAT 추가
  - `RUN_APP.bat`
  - `RUN_APP_FAST.bat`
  - `INSTALL_AND_RUN.bat`

- 프로젝트 빠른 복구 메뉴 추가
  - 이전 DB가 없어도 쿠팡 링크 + 상품명 + 원본 영상으로 프로젝트를 다시 생성
  - 타오바오 TOP10 / 도우인 TOP10 검색어 자동 생성
  - 생성 후 원클릭 파이프라인으로 바로 이어갈 수 있음

- 데이터 가져오기 개선
  - 자동 감지 후보를 라디오 버튼으로 선택 가능

## 실행 방법
1. 압축 해제
2. `RUN_APP.bat` 더블클릭
3. 브라우저에서 앱 실행 확인

## 예전 DB가 없을 때
1. `🧩 프로젝트 빠른 복구`
2. 쿠팡 링크 입력
3. 상품명 붙여넣기
4. 원본 영상 선택
5. 프로젝트 복구/생성
6. `⚡ 원클릭 파이프라인` 실행
