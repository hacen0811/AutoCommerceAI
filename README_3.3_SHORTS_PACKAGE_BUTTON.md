# AutoCommerceAI 3.3 Shorts Package Button

## 개선 내용
- 쇼츠 스튜디오의 `쇼핑쇼츠 패키지 생성` 버튼 결과 저장 기능 추가
- 결과를 JSON/TXT/MD 파일로 자동 저장
- 저장 위치: `exports/shorts_packages`
- 다음 작업 체크리스트 표시
- 유튜브/인스타/인포크 문구까지 파일로 저장

## 실행
`RUN_APP.bat` 더블클릭

## 테스트 순서
1. 쇼츠 스튜디오 3.0 클릭
2. 쿠팡 링크 입력
3. 상품명 입력
4. 원본 영상 선택
5. 쇼핑쇼츠 패키지 생성 클릭
6. 검색어/대본/CapCut/업로드 문구와 저장 파일 확인
