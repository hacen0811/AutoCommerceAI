# AutoCommerceAI 1.8.1 Data Keep Patch

## 핵심 수정
새 버전 폴더를 실행하면 프로젝트가 없어지는 문제를 해결하기 위해 `📦 데이터 가져오기` 메뉴를 추가했습니다.

## 사용 방법
1. 1.8.1 실행
2. 왼쪽 메뉴 `📦 데이터 가져오기`
3. 이전 버전 폴더 경로 입력
4. `이전 데이터 가져오기` 클릭
5. 새로고침 또는 재실행
6. 원클릭 파이프라인에서 기존 프로젝트 확인

## 가져오는 데이터
- `database/autocommerce_x.db`
- `assets/videos`
- `assets/images`
- 선택 시 `exports`
