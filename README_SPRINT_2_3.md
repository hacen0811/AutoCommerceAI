# Sprint 2.3 - Source Video AI

## 완료 내용
- Source Video AI 메뉴 추가
- 타오바오/도우인 검색 계획 생성
- 플랫폼별 소스 URL 등록
- 사용자가 확보한 원본 영상 프로젝트별 저장
- 영상 후보 빠른 랭킹
- Batch Analyze 준비
- Real Vision Runner 연계

## 중요한 기준
타오바오/도우인 영상 자동 다운로드는 각 서비스 약관과 저작권을 확인해야 하므로,
이번 버전은 사용자가 직접 확보한 원본 영상을 등록하고 분석하는 방식으로 구현했습니다.

## 사용 순서
1. Product AI 2.0에서 프로젝트 생성
2. Source Video AI에서 검색어 확인
3. 직접 확보한 원본 영상 업로드
4. 후보 랭킹 실행
5. Batch Analyze 또는 Real Vision Runner 실행
